RESTful Routes

A table of all 7 RESTful routes

Name Url HTTP Verb Purpose Mongoose Method
INDEX /dogs GET Display a list of all dogs Dog.find( )
NEW /dogs/new GET Displays form to add a new dog N/A
CREATE /dogs POST Add a new dog to database, redirect somewhere Dog.create( )
SHOW /dogs/:id GET Shows info about one dog Dog.findById( )
EDIT /dogs/:id/edit GET Show edit form for one dog Dog.findById( )
UPDATE /dogs/:id PUT Update a particular dog then redirect somewhere Dog.findByIdAndUpdate( )
DESTROY /dogs/:id DELETE Delete a particular dog then redirect somewhere Dog.findByIdAndRemove( )
